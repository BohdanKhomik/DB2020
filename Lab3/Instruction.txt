Для роботи програми використовується flask та psycopg2
Потрбіно інсталювати їх, в термінал вводимо 
$ pip install psycopg2
$ pip install Flask
Після запуску проекту заходимо на http://127.0.0.1:5000/
Опиняємося на головній сторінці, тут обираємо табличку, з якою хочемо працювати, після цього опиняємося на сторінці де потрібно ввести ID елементу, який хочемо переглянути, після цього стають доступними функції UPDAT та
DELETE, та можливість ввести інше ID елемента. Також є функція CREATE, яка створює новий запис у таблиці.
