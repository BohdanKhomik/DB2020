Для роботи програми використовується flask та psycopg2
Потрбіно інсталювати їх, в термінал вводимо 
$ pip install psycopg2
$ pip install Flask
Після запуску проекту заходимо на http://127.0.0.1:5000/
Опиняємося на головній сторінці, тут вводимо ID елементу, який хочемо переглянути, після цього стають доступними функції UPDATE,
DELETE та CREATE, та можливість ввести інше ID елемента.
